# Week 1 - Linux Basics

## What I learned today
- What a terminal is
- Basic navigation in Linux
- How files and directories work

## Commands practiced
- pwd → shows current directory
- ls → lists files
- cd → change directory
- mkdir → create folder
- touch → create file

## Notes
Linux is the foundation of DevOps because most servers run Linux.  
Learning the terminal is more important than clicking around with a mouse.

## Goal
Get comfortable navigating without a GUI.
